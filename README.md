## 🚀 О проекте

KÖRPÜ (мост) — это полнофункциональный мессенджер, работающий как Telegram Web App так и как  приложение (Андроид). Два способа входа: регистрация с сохранением профиля или анонимный чат без следов.

## ✨ Возможности

### 🔐 Авторизация
- Регистрация с логином и паролем (хранятся в PostgreSQL)
- Быстрый анонимный чат — только никнейм, без пароля (Рандомный логин удаляется через 1 час)
- Автоматический вход при повторном визите
- Защита от одновременных сессий

### 💬 Чат
- Личные сообщения в реальном времени (WebSocket)(P2P)
- Групповые чаты — объединяй друзей в группы
- Вкладки чатов — переключайся между диалогами
- Голосовые сообщения
- Эмодзи-панель (40+ эмодзи)
- Реакции на сообщения (👍❤️😂😮😢🔥)
- Индикатор «печатает...»
- Доставка ✓ и прочтение ✓✓

### 👫 Друзья
- Добавление и удаление друзей
- Список друзей с онлайн-статусом
- Быстрое подключение к другу из списка
- Пометка «друг» в списке онлайн

### 👤 Профиль
- 16 аватаров-животных (🐱🐶🦊🐼🐨🦁🐯🐸🐵🦄🐺🐰🐻🦋🐙🐧)
- Статусы: Онлайн, Отдыхаю, Ку-Ку, Он тут, Она тут, Занят, Играю
- Быстрое копирование своего ID

### 🎨 Интерфейс
- Тёмная и светлая тема
- Glassmorphism дизайн с анимациями
- Полноэкранный режим в Telegram (если использовать его как Telegram web app)
- Звуковые уведомления
- Кнопка UPDATE с changelog

### 🕶 Анонимный режим
- Вход без регистрации
- Не видит список онлайн пользователей
- Не видит систему друзей
- Полный доступ к чату
